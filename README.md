# registration-backend
Backend for user registration using Flask
## Registration Backend

Backend for user registration built using Flask.

### Features
- User registration API
- Input validation
- JSON-based responses

### Tech Stack
- Python (Flask)
- GitHub

### API Endpoint
POST /register

### Example Request

POST /register

{
  "name": "Tanvi",
  "email": "tanvi@gmail.com",
  "password": "123456"
}
